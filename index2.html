Cflow

Cflow is a BPM (Business Process Management : Workflow) suite for enterprise use which contains:

Bonita BPM improves business operations by connecting people, processes, and information systems into easily managed applications. Download the most widely adopted BPM solution in the world.
  • A workflow engine that drives process automation and work distribution
  • A set of OpenAPI and SDK for developer to develop process-oriented applications
  • A workflow designer for designing workflow template with GUI
  • A process monitor to check process running status
  • A developer console to control and config workflow context

Installation:

Cflow has been architected to be high avalible, the single-server mode deployment works well for most small to medium organization, the multi-server deployment can be used in higher load environment or running CFlow in "Workflow as a Service" mode.

You may donwload Cflow package, install and run it immediately (installation guide is here for your reference).

Integration:

You may use developer console to config cflow business context (see developer console instruction)

Use Cflow OpenAPI or SDK to integrate BPM/Workflow function into your own application, see integration instruction here.
