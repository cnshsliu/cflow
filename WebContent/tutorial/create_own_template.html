Create Workflow Template

To creat workflow template, you need to use WFT demo site.

Click "Template Library" in WFT demo site menu, The workflow designer will be displayed.

Toolbars:

ToolDescription Properties

Toggle to SELECT mode.

In this mode:

  1. Click on a node to select it
  2. Select and hold a node to move it
  3. Double click a node to show node properties
  4. Select and hold a link then drag to change link ends.
  5. Double click on a link to show link properties
  

Toggle to LINK mode

In this mode

  1. Click on one node (the preceding task)
  2. Click on another node (the following task),
  3. An arrow will be draw between these two nodes.
  

Create a new TASK node.

A TASK node indicates a task which should be done by one or more person.

 see task properties

Create a new TIMER node

A TIMER node will pause until specified period expires, is used to hold process for a certain period.

 see timer properties

Create a new Sub Process Node

Embed a process into current process.

 see sub process properties

Create a SCRIPT node

A SCRIPT node contains codes to be run.

 see script properties

Create an AND node

An AND node will be passed automatically when all its precedding tasks complete.

  

Create an OR node

An OR node will be passed automatically when any one of its precedding tasks completes

  

Create an ROUND node.

An ROUND node indicates a group of nodes can be run many times, and the contect values (for example, user input) will be kept for each round.

 see round properties

Create an GROUND node

An GROUND node have no following nodes.

  

Toggle to DELETE mode

Once in DELETE mode, click on a node or a link to delete it.

  

Create a new template

Start a new template, the current change will be dismissed.

  

Save the current template

Save the current template

  

Maximize button

Maximize the workflow designer to the size of browser window.
