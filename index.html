cflow

Cflow is a BPM (Business Process Management : Workflow) suite which contains: A workflow engine that drives process automation and work distribution A set of OpenAPI and SDK for developer to develop process-oriented applications A workflow designer for designing workflow template with GUI A process monitor to check process running status A developer console to control and config workflow context

Cflow has been architected to be high avalible, the single-server mode deployment works well for most small to medium organization, the multi-server deployment can be used in higher load environment or running CFlow in "Workflow as a Service" mode.

Cflow's source codes are hosted on Github, if you are intrested in contributing code to this project, see contributor guide here.

You may donwload Cflow package, install and run it immediately (installation guide is here for your reference), after installation, you may use developer console to config cflow business context (see developer console instruction)

Use Cflow OpenAPI or SDK to integrate BPM/Workflow function in any application, see integration instruction here.
